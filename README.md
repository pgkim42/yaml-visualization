# YAML Visualizer

YAML 파일을 구문 강조와 함께 시각적으로 보여주고, 실시간으로 편집할 수 있는 웹 기반 도구

## 주요 기능

- **YAML 에디터**: 구문 강조, 자동 들여쓰기, 실시간 유효성 검사
- **트리 뷰어**: 계층 구조를 접기/펼치기 가능한 트리로 표시
- **양방향 동기화**: 에디터 수정 시 뷰어 실시간 업데이트
- **유틸리티**: 파일 업로드/다운로드, 클립보드 복사, JSON 변환

## 기술 스택

- React 18 + TypeScript
- Vite
- Monaco Editor
- Zustand
- Tailwind CSS

## 시작하기

```bash
npm install
npm run dev
```
