# YAML Visualization - Product Requirements Document

## 1. 프로젝트 개요

### 1.1 제품명
YAML Visualizer

### 1.2 목적
YAML 파일을 구문 강조와 함께 시각적으로 보여주고, 실시간으로 편집할 수 있는 웹 기반 도구

### 1.3 대상 사용자
- 개발자
- DevOps 엔지니어
- 시스템 관리자
- YAML 설정 파일을 다루는 모든 사용자

---

## 2. 핵심 기능

### 2.1 YAML 에디터
- **구문 강조(Syntax Highlighting)**: 키, 값, 문자열, 숫자, 불리언 등 타입별 색상 구분
- **자동 들여쓰기**: YAML 규칙에 맞는 자동 들여쓰기
- **라인 번호 표시**: 코드 라인 번호 표시
- **실시간 유효성 검사**: YAML 문법 오류 실시간 감지 및 표시

### 2.2 시각화 뷰어
- **트리 구조 표시**: YAML 계층 구조를 접기/펼치기 가능한 트리로 표시
- **타입 표시**: 각 값의 데이터 타입 시각적 표시 (string, number, boolean, array, object)
- **검색 기능**: 키 또는 값 검색
- **경로 표시**: 선택한 항목의 전체 경로 표시

### 2.3 양방향 동기화
- 에디터 수정 시 뷰어 실시간 업데이트
- 뷰어에서 값 클릭 시 에디터 해당 위치로 이동

### 2.4 유틸리티 기능
- **파일 업로드**: 로컬 YAML 파일 불러오기
- **다운로드**: 편집한 YAML 파일 다운로드
- **복사**: 클립보드에 복사
- **포맷팅**: YAML 자동 정렬/포맷팅
- **JSON 변환**: YAML ↔ JSON 상호 변환

## 3. UI/UX 요구사항

### 3.1 레이아웃
```
+--------------------------------------------------+
|                    헤더 (로고, 메뉴)               |
+------------------------+-------------------------+
|                        |                         |
|      YAML 에디터        |      시각화 뷰어         |
|      (좌측 패널)        |      (우측 패널)         |
|                        |                         |
+------------------------+-------------------------+
|               하단 상태바 (에러/경고 표시)          |
+--------------------------------------------------+
```

### 3.2 반응형 디자인
- 데스크톱: 좌우 분할 뷰
- 태블릿/모바일: 탭으로 전환

### 3.3 테마
- 라이트 모드
- 다크 모드

## 4. 비기능적 요구사항

### 4.1 성능
- 초기 로딩: 3초 이내
- 에디터 입력 반응: 100ms 이내
- 대용량 YAML (10,000줄 이상) 처리 가능

### 4.2 브라우저 호환성
- Chrome (최신 버전)
- Firefox (최신 버전)
- Safari (최신 버전)
- Edge (최신 버전)

### 4.3 접근성
- 키보드 네비게이션 지원
- 스크린 리더 호환

---

## 5. 기술 스택 (권장)

### 5.1 프론트엔드
- **프레임워크**: React 19 with TypeScript
- **빌드 도구**: Vite
- **에디터**: Monaco Editor (VSCode 에디터 기반) 또는 CodeMirror 6
- **YAML 파서**: yaml (js-yaml 대체)
- **스타일링**: CSS Modules 또는 Tailwind CSS
- **상태 관리**: Zustand (경량) 또는 React Context

### 5.2 주요 라이브러리
- `yaml`: YAML 파싱 및 직렬화
- `@monaco-editor/react`: Monaco 에디터 React 래퍼
- `react-arborist` 또는 커스텀 트리 컴포넌트: 트리 뷰 구현
